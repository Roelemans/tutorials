March Tutorials¶

Welcome to the main documentation on the March IV exoskeleton. Whether you are an experienced ROS developer or have recently started learning, these tutorials contain all knowledge required to work with and develop the March IV exoskeleton.

Requirements¶

In order to follow the tutorials, we expect you to have at least some knowledge of the following tools:

  • Ubuntu 16.04
  • Linux command line
  • Git

In case you are not comfortable with the above tools, please take some time to check the following references:

  • Git & GitHub Crash Course For Beginners
  • Linux command line for beginners

How to use the tutorials¶

The tutorials are designed to provide hands-on experience with ROS and the March IV exoskeleton. Here is an overview of the different types of tutorials.

Create your workspace¶

This tutorial will help you set up a ROS workspace with all packages needed to run the March IV exoskeleton.

Intro to ROS¶

This tutorial will introduce you to the world of ROS by referring to the official ROS documentation.

Useful ROS tools¶

A selection of existing ROS tools to speed up development.

March Packages¶

This section contains explanation and tutorials at the package level. It is not needed to study all of them to completion, but it is advised to keep them as a reference whenever you want start improving a specific package. This is especially useful when adding more content to a feature (e.g. gaits, launch settings).

Documentation¶

This tutorial will teach you how to build the documentation locally and contribute to its development.
